# 🐑 數綿羊 - 助眠小工具

一個簡單、舒緩的「數綿羊」網頁應用程式，幫助你在睡不著的時候放鬆入夢。

## 功能

- 🎯 **自動計數**：綿羊一隻隻跳出來，幫你數數
- 🔊 **語音數羊**（可選）：用 TTS 讀出綿羊數量，支援中文、英文、日文
- ⚙️ **可調速度**：從每 0.5 秒到 5 秒一隻，自由調節
- 🌙 **暗色主題**：保護眼睛，適合睡前使用
- 📱 **響應式設計**：手機、平板、電腦都能用

## 使用方式

1. 開啟 `index.html`（可以用瀏覽器直接開啟）
2. 點「▶ 開始數」按鈕
3. 閉上眼睛，聽著/看著綿羊慢慢數
4. 調整速度或語音選項，找到最適合你的節奏

## 語音設定

首次使用語音功能，瀏覽器可能會詢問是否允許語音合成，請點「允許」。

選擇語言後，綿羊數量會用對應的語言朗讀（如：`10隻綿羊`、`ten sheep`、`10匹の羊`）。

## 為什麼有效？

數綿羊是一種經典的放鬆技巧，透過重複、單調的視覺/聽覺刺激，讓大腦從雜念中解脫，慢慢進入放鬆狀態，進而自然入睡。
